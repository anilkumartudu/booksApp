# booksApp
Simple book app for Android written with React Native.

# Try it yourself
---

## 1.Clone and Install
---
```bash
# Clone the repo
git clone https://github.com/aniltudu/booksApp.git

# Open the folder and Install dependencies.
cd booksApp && npm install
```

## 2. Run it on Android
---
```bash
react-native run-android
```
